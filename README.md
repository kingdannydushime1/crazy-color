# Crazy Color

A color mixing puzzle game built with Phaser 4 and React.

Mix primary colors (red, blue, yellow) to match target colors and progress through levels. Features CrazyGames SDK integration for rewarded video ads.

## Development

```bash
npm install
npm run dev
```

Server runs on `http://localhost:8000`.

## Controls

- Tap buttons 1/2/3 on screen (or keyboard 1/2/3) to dispense color drops
- Mix drops in the flask to match the target color
- Earn coins, buy skins and themes from the shop
- Watch rewarded ads for refills, undos, revives, and bonus coins

Built with [Phaser 4](https://phaser.io/).
